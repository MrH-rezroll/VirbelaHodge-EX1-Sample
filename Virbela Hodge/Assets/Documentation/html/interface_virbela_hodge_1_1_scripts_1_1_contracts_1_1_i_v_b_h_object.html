Detailed Description

This Interface is a Contract that forces all implementing classes to be "Virbela Hodge Objects" (vBHObjects, vBHOs). All objects implementing this interface are usable within the business logic of this example.

Generates and randomly places a specified number of VBHObjects of a specified type. This generator accounts for any VBHOs manually placed within the scene and will only generate the difference between the number of VBHOs of the specified type that exist, and the max amount that can exist.

Parameters
vBHObjectsThe list of VBHObjects that will be added to.
prefabToGenerateThe prefab of the type of VBHO to generate.
maxAmountThe total amount of the specified type that should exist.
rndSeedThe random seed value for this type of VBHO.
